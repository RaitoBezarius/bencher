### CLI-Bibliothek

Beim Erstellen einer Rust-CLI ist [Clap][github clap] ein wenig der de facto Standard. Stellen Sie sich also meinen Schock vor, als [ich Bencher zum ersten Mal öffentlich vorführte][talks rust dc] und der Schöpfer selbst, [Ed Page][github epage], dort war! 🤩

Im Laufe der Zeit entdecke ich immer mehr nützliche Dinge, die Clap tun kann. Es ist ein wenig peinlich, aber ich habe erst kürzlich [die `default_value`-Option][docs clap default value] entdeckt. All diese Fähigkeiten helfen wirklich, die Menge des Codes zu reduzieren, den ich in der `bencher`-CLI pflegen muss.

> 🐰 Fun Fact: [Clap verwendet Bencher][clap github pr 5826], um die [Binärgröße zu verfolgen][track binary size]!
