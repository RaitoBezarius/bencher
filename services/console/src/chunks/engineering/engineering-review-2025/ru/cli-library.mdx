### CLI Библиотека

При создании CLI на Rust, [Clap][github clap] является своего рода стандартом де факто. Представьте мое удивление, когда [я впервые публично демонстрировал Bencher][talks rust dc], и сам создатель, [Ed Page][github epage], был там! 🤩

Со временем я продолжаю находить все больше полезных вещей, которые может делать Clap. Это немного стыдно, но я только недавно обнаружил [опцию `default_value`][docs clap default value]. Все эти возможности действительно помогают сократить количество кода, который я должен поддерживать в `bencher` CLI.

> 🐰 Интересный факт: [Clap использует Bencher][clap github pr 5826] для [отслеживания размера бинарных файлов][track binary size]!
