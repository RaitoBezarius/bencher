### Biblioteca CLI

Ao construir uma CLI em Rust, [Clap][github clap] é um tipo de padrão de fato. Então imagine meu choque quando [eu fiz a primeira demonstração pública do Bencher][talks rust dc] e o próprio criador, [Ed Page][github epage], estava lá! 🤩

Com o tempo, continuo descobrindo mais e mais coisas úteis que o Clap pode fazer. É um pouco constrangedor, mas acabei de descobrir [a opção `default_value`][docs clap default value]. Todas essas capacidades realmente ajudam a reduzir a quantidade de código que tenho que manter no CLI `bencher`.

> 🐰 Curiosidade: [Clap usa Bencher][clap github pr 5826] para [monitorar o tamanho do binário][track binary size]!
