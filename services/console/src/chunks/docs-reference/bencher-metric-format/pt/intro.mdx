Os adaptadores de [harness de benchmark integrados][adapters] convertem a saída do benchmark para o formato JSON do Bencher Metric Format (BMF).
Harnesses de benchmark personalizados devem emitir seus resultados como BMF JSON
e usar [o adaptador `json`][json adapter].
Veja [como acompanhar benchmarks personalizados][custom benchmarks] para uma visão geral completa.
