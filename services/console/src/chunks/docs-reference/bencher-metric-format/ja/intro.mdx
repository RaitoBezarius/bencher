内蔵された[ベンチマークハーネスアダプター][adapters]は、ベンチマークの出力をBencher Metric Format (BMF) JSONに変換します。
カスタムベンチマーキングハーネスは、その結果をBMF JSONとして出力し、[`json`アダプター][json adapter]を使用すべきです。
完全な概要については、[カスタムベンチマークの追跡方法][custom benchmarks]をご覧ください。
