내장된 [벤치마크 하네스 어댑터][adapters]는 벤치마크 출력을 Bencher Metric Format (BMF) JSON으로 변환합니다.
사용자 정의 벤치마킹 하네스는 BMF JSON으로 결과를 출력하고 [json 어댑터][json adapter]를 사용해야 합니다.
전체 개요에 대해서는 [사용자 정의 벤치마크 추적 방법][custom benchmarks]을 참조하세요.
