内置的[基准测试适配器][adapters]会将基准测试的输出转换为 Bencher Metric Format (BMF) JSON 格式。
自定义基准测试工具应输出其结果为 BMF JSON 格式，
并使用[`json` 适配器][json adapter]。
查看[如何跟踪自定义基准测试][custom benchmarks]以获得完整概述。
