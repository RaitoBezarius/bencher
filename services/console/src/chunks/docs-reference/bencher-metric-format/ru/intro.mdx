Встроенные [адаптеры для тестовых стендов][adapters] преобразуют вывод результатов тестирования в формат JSON формата метрик Bencher (BMF).
Пользовательские тестовые стенды должны выводить свои результаты в формате BMF JSON
и использовать [адаптер `json`][json adapter].
Смотрите [как отслеживать пользовательские бенчмарки][custom benchmarks] для полного обзора.
