Les adaptateurs de harnais de benchmark intégrés convertissent la sortie de benchmark en format JSON de métriques Bencher (BMF). Les harnais de benchmarking personnalisés devraient produire leurs résultats en JSON BMF et utiliser [l'adaptateur `json`][json adapter]. Consultez [comment suivre des benchmarks personnalisés][custom benchmarks] pour un aperçu complet.
