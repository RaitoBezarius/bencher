Integrierte [Benchmark-Harness-Adapter][adapters] wandeln Benchmark-Ergebnisse in das Bencher Metric Format (BMF) JSON um.
Benutzerdefinierte Benchmarking-Harnesses sollten ihre Ergebnisse als BMF JSON ausgeben
und [den `json` Adapter][json adapter] verwenden.
Siehe [wie man benutzerdefinierte Benchmarks verfolgt][custom benchmarks] für einen vollständigen Überblick.
