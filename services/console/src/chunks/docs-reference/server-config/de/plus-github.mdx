### `plus.github`

Dieser Abschnitt spezifiziert die Konfiguration für [eine GitHub App, die für die OAuth2-Authentifizierung verwendet wird](/de/docs/how-to/github-app/).
Sie müssen eine gültige Bencher Plus _Enterprise_ Lizenz für mindestens eine Organisation auf dem Server haben.
Der gesamte Abschnitt ist optional.
Wenn er nicht angegeben ist, wird die Authentifizierung mit GitHub nicht aktiviert.
