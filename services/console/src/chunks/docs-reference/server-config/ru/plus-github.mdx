### `plus.github`

В этом разделе указывается конфигурация [приложения GitHub, используемого для аутентификации OAuth2](/ru/docs/how-to/github-app/).
У вас должна быть действующая лицензия Bencher Plus _Enterprise_ для хотя бы одной организации на сервере.
Весь раздел является необязательным.
Если он не указан, то аутентификация с помощью GitHub не будет включена.
