### `plus.rate_limiting`

Cette section spécifie la limitation de débit pour le serveur API.
Toute la section est facultative.
Si elle n'est pas spécifiée, les valeurs par défaut énumérées seront utilisées.

Pour empêcher la création de projets [`unclaimed`][unclaimed projects], définissez la `unclaimed_limit` à `0`.

|       Nom       | Exemple | Défaut | Requis |                                                                             Description                                                                              |
| :-------------: | :-----: | :----: | :----: | :------------------------------------------------------------------------------------------------------------------------------------------------------------------: |
|     window      |  86400  | 86400  |  Non   |                                       Spécifie la période de limitation de débit en secondes. Défini par défaut à une journée.                                       |
|   user_limit    |   255   |  255   |  Non   |                          Contrôle la limitation de débit pour certaines ressources créées par un utilisateur : Organisations et jetons API.                          |
| unclaimed_limit |   255   |  255   |  Non   | Contrôle la limitation de débit pour les projets [`unclaimed`][unclaimed projects]. Pour empêcher la création de projets `unclaimed`, définissez cette valeur à `0`. |
|  claimed_limit  |  65536  | 65536  |  Non   |                                                     Contrôle la limitation de débit pour les projets `claimed`.                                                      |

[unclaimed projects]: /fr/docs/how-to/claim/#unclaimed-projects
