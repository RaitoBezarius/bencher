### `plus.github`

Esta seção especifica a configuração para [um aplicativo GitHub usado para autenticação OAuth2](/pt/docs/how-to/github-app/).
Você deve ter uma licença _Enterprise_ válida do Bencher Plus para pelo menos uma organização no servidor.
A seção inteira é opcional.
Se não especificada, a autenticação com o GitHub não será ativada.
