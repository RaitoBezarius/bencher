### `plus.github`

이 섹션은 [OAuth2 인증을 위해 사용되는 GitHub 앱](/ko/docs/how-to/github-app/)의 설정을 지정합니다.
서버에 최소 한 개의 조직이 Bencher Plus _Enterprise_ 라이센스를 유효하게 가져야 합니다.
전체 섹션은 선택 사항입니다.
지정되지 않으면 GitHub을 사용한 인증이 활성화되지 않습니다.

|     이름      |                  예시                   | 기본값 | 필수  |                                                                                                                                                                                    설명                                                                                                                                                                                    |
| :-----------: | :-------------------------------------: | :----: | :---: | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------: |
|   client_id   |          Iv1.12864abcd1232048           |  ---   |  예   | GitHub 앱에 대한 클라이언트 ID를 지정합니다. 클라이언트 ID는 앱 ID와 다릅니다. 클라이언트 ID는 앱의 설정 페이지에서 찾을 수 있습니다. GitHub 앱 설정 페이지로 이동하는 방법에 대한 자세한 정보는 [GitHub 앱 등록 수정](https://docs.github.com/en/apps/maintaining-github-apps/modifying-a-github-app-registration#navigating-to-your-github-app-settings)를 참조하십시오. |
| client_secret | 00000abcd12345wxyz123456789abcdefgh0000 |  ---   |  예   |                                                                                                                                   GitHub 앱에 대한 클라이언트 시크릿입니다. 앱의 설정 페이지에서 클라이언트 시크릿을 생성할 수 있습니다.                                                                                                                                   |
