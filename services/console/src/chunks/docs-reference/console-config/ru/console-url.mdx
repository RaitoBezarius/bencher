## URL консоли

По умолчанию [сервер API Bencher][server config] ожидает, что сервер Bencher Console будет запущен по адресу `http://localhost:3000`.
Если ваш сервер Bencher Console запущен в другом месте, вам необходимо установить
[опцию `console.url`][console url].
