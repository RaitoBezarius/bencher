## HTTPS

По умолчанию сервер Bencher Console использует HTTP.
Это хорошо работает, если у вас есть прокси-сервер перед ним для обработки HTTPS.
Если вам нужен автономный сервер Console для самостоятельной обработки HTTPS,
то вам необходимо предоставить SSL-ключ и сертификат.

Вы можете настроить HTTPS, используя следующие переменные окружения:
  - `SERVER_KEY_PATH`: Путь к файлу вашего TLS-ключа.
  - `SERVER_CERT_PATH`: Путь к файлу вашего TLS-сертификата.
