## URL de la Console

Par défaut, [le serveur API de Bencher][server config] s'attend à ce que le serveur Console de Bencher fonctionne à `http://localhost:3000`.
Si votre serveur Console de Bencher fonctionne ailleurs, vous devez alors configurer
[l'option `console.url`][console url].
