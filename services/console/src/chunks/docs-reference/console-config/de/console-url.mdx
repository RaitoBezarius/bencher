## Konsolen-URL

Standardmäßig erwartet [der Bencher API-Server][server config], dass der Bencher Konsolen-Server unter `http://localhost:3000` läuft.
Falls Ihr Bencher Konsolen-Server woanders läuft, müssen Sie die
[Option `console.url`][console url] einstellen.

[console url]: /de/docs/reference/server-config/#console
