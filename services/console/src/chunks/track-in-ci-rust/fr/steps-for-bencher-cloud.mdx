## Étapes pour Bencher Cloud

1. [Créez un compte Bencher Cloud][bencher sign up].
2. [Créez un jeton API][console users tokens add] et ajoutez-le à votre CI en tant que secret.
3. Créez un workflow pour votre CI, comme [GitHub Actions][github actions] ou [GitLab CI/CD][gitlab ci cd].
4. [Installez le Bencher CLI][install bencher cli] dans votre workflow CI.
5. Exécutez vos benchmarks avec la
   <code><a href="/fr/docs/explanation/bencher-run/">sous-commande bencher run</a></code>
   dans votre workflow CI en utilisant
   <a href={props.url}>l'adaptateur <code>{props.adapter}</code></a>.
