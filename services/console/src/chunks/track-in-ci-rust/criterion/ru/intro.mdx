Теперь, когда вы узнали [как проводить бенчмаркинг кода на Rust с помощью Criterion][learn criterion],
давайте посмотрим, как отслеживать эти бенчмарки в CI.
[Непрерывный бенчмаркинг][continuous benchmarking] — это практика запуска бенчмарков для каждого набора изменений, чтобы убедиться, что изменения не вводят регрессии производительности.
Самый простой способ реализовать Непрерывный бенчмаркинг с бенчмарками Criterion — использовать Bencher.
