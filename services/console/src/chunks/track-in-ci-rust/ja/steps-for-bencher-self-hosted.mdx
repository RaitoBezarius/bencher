## Bencher Self-Hosted の手順

1. [Bencher Self-Hosted インスタンスを作成します][bencher self hosted]。
2. Bencher Self-Hosted インスタンスでアカウントを作成します。
3. API トークンを作成し、それをシークレットとして CI に追加します。
4. [GitHub Actions][github actions] や [GitLab CI/CD][gitlab ci cd] など、CI のワークフローを作成します。
5. CI ワークフローで [Bencher CLI をインストール][install bencher cli] します。CLI のバージョンが Bencher Self-Hosted インスタンスのバージョンと一致していることを確認してください。
6. 次のサブコマンドを使用して CI ワークフローでベンチマークを実行します:
   <code><a href="/ja/docs/explanation/bencher-run/">bencher run</a></code>
   <a href={props.url}><code>{props.adapter}</code> アダプター</a>を使用して、
   [ `--host` オプション][host option] を Bencher Self-Hosted インスタンスの URL に設定します。
