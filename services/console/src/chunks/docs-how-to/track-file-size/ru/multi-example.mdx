## Несколько размеров файлов

Если вы хотите отслеживать размеры нескольких файлов,
вы можете просто использовать опцию `--file-size` несколько раз в одной и той же
<code><a href="/ru/docs/explanation/bencher-run/">bencher run</a></code> команде.
Например, вы можете отслеживать как `my_binary`, так и версию для Windows, `my_binary.exe`.
