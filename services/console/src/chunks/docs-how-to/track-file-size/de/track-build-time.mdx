## Buildzeit verfolgen

Sie möchten möglicherweise [die Buildzeit verfolgen][track build time] Ihrer Liefergegenstände,
zusätzlich zur Verfolgung ihrer Dateigröße.
<code><a href="/de/docs/explanation/bencher-run/">bencher run</a></code>
unterstützt sowohl die Verfolgung von Dateigrößen als auch von Buildzeiten.
