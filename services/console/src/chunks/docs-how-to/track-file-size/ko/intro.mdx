Bencher는 기본적으로 [가장 인기 있는 벤치마킹 하네스][adapters]를 지원합니다.
그러나 때로는 실행 가능 파일의 이진 크기와 같이, 배포물 자체의 크기를 측정하고 싶을 수 있습니다.
여러분에게 행운이지만, Bencher는 파일 크기 추적도 지원합니다.
파일 크기를 추적하는 가장 쉬운 방법은
<code><a href="/ko/docs/explanation/bencher-run/">bencher run</a></code> CLI 하위 명령어를
`--file-size` 옵션과 함께 사용하는 것입니다.
`--file-size` 옵션은 크기가 측정될 파일의 파일 경로를 기대합니다.
내부적으로, <code><a href="/ko/docs/explanation/bencher-run/">bencher run</a></code>
은 결과를 [Bencher Metric Format (BMF) JSON][bmf]으로 출력합니다.
따라서 명시적으로 [the `json` adapter][json adapter]를 사용하는 것이 좋은 관행입니다.
