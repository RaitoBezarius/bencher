## Seguimiento del Tiempo de Compilación

Puede que desees [realizar un seguimiento del tiempo de compilación][track build time] de tus entregables, además de seguir su tamaño de archivo.
<code><a href="/es/docs/explanation/bencher-run/">bencher run</a></code>
admite el seguimiento tanto de los tamaños de archivo como de los tiempos de compilación.
