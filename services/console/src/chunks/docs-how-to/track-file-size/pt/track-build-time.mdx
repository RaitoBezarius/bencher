## Acompanhar Tempo de Compilação

Você pode querer [acompanhar o tempo de compilação][track build time] de seus entregáveis, além de acompanhar o tamanho de seus arquivos. 
<code><a href="/pt/docs/explanation/bencher-run/">bencher run</a></code>
suporta o acompanhamento tanto dos tamanhos dos arquivos quanto dos tempos de compilação.
