## 多文件大小

如果你想跟踪多个文件的大小，你只需在同一个
<code><a href="/zh/docs/explanation/bencher-run/">bencher run</a></code> 命令中多次使用 `--file-size` 选项。
例如，你可以跟踪 `my_binary` 和一个 Windows 可执行文件版本 `my_binary.exe`。
