## GitHub Actions

[GitHub Action](https://github.com/marketplace/actions/bencher-cli)을 사용하여 Bencher CLI를 설치하세요.
자세한 내용은 [GitHub Actions 사용법](/ko/docs/how-to/github-actions/)을 참조하세요.

<CliGitHubActions />

사용할 Bencher CLI의 정확한 버전을 지정하는 선택적인 `version` 인수가 있습니다.
그렇지 않으면 최신 CLI 버전을 사용하는 것으로 기본 설정됩니다.
Bencher *Self-Hosted*를 사용하는 경우에만 정확한 버전을 지정하세요.
아직 끊임없는 주요 변경 사항이 있기 때문에 Bencher *Cloud*를 사용하는 경우에는 정확한 버전을 지정하지 __마세요__.
