## Seguimiento del Tamaño de Archivo

Es posible que desee [hacer un seguimiento del tamaño del archivo][track file size] de sus entregables, además de seguir cuánto tiempo tardan en compilarse. <code><a href="/es/docs/explanation/bencher-run/">bencher run</a></code> admite el seguimiento tanto de los tiempos de compilación como de los tamaños de archivo.
