Bencher admite [los arneses de evaluación comparativa más populares][adapters] de manera predeterminada. Sin embargo, a veces deseas medir cuánto tiempo lleva compilar tus entregables, como el tiempo de compilación de un ejecutable. Afortunadamente, Bencher también admite el seguimiento de los tiempos de compilación. La forma más fácil de rastrear los tiempos de compilación es usar el subcomando CLI <code><a href="/es/docs/explanation/bencher-run/">bencher run</a></code> con el flag `--build-time`. Internamente, <code><a href="/es/docs/explanation/bencher-run/">bencher run</a></code> entrega los resultados como [Bencher Metric Format (BMF) JSON][bmf]. Por lo tanto, es una buena práctica usar explícitamente [el adaptador `json`][json adapter].
