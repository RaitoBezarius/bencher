## Suivre la Taille des Fichiers

Vous pouvez vouloir [suivre la taille des fichiers][track file size] de vos livrables, en plus de suivre le temps nécessaire pour les construire. <code><a href="/fr/docs/explanation/bencher-run/">bencher run</a></code> prend en charge le suivi des temps de construction et des tailles de fichiers.
