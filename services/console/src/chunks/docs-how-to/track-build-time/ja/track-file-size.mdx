成果物のビルドにかかる時間を追跡することに加えて、[ファイルサイズを追跡する][track file size] かもしれません。
<code><a href="/ja/docs/explanation/bencher-run/">bencher run</a></code>
は、ビルド時間とファイルサイズの両方を追跡することをサポートしています。
