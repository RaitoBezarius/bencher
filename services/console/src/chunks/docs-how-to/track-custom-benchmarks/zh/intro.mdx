Bencher 默认支持[最受欢迎的基准测试工具][adapters]，而且我们始终[欢迎建议][github issue]添加新的适配器。
然而，在某些情况下，现成的基准测试工具可能无法满足您的需求，
这就需要创建自定义的基准测试工具。
幸运的是，Bencher 也支持使用自定义的基准测试工具。
将您的自定义基准测试工具与 Bencher 集成的最简单方式
是输出 [Bencher Metric Format (BMF) JSON][bmf]。
