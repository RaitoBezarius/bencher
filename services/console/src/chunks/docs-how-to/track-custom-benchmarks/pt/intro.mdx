Bencher suporta [os dispositivos de benchmarking mais populares][adapters] prontamente,
e estamos sempre [abertos a sugestões][github issue] para novos adaptadores.
No entanto, podem ocorrer situações em que um dispositivo de benchmarking pronto para uso não atenda às suas necessidades,
necessitando a criação de um dispositivo personalizado de benchmarking.
Felizmente para você, o Bencher também suporta o uso de um dispositivo de benchmarking personalizado.
O jeito mais fácil de integrar seu dispositivo de benchmarking personalizado com o Bencher
é produzir [Bencher Metric Format (BMF) JSON][bmf].
