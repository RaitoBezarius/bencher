Depending on your use case, you can set up [Continuous Benchmarking][continuous benchmarking] in GitLab CI/CD for your:

- [Target Branch](#target-branch)
- [Merge Requests](#merge-requests)

Stellen Sie sicher, dass Sie ein [API-Token erstellt][created an api token] haben und es als maskierte Variable mit dem Namen `BENCHER_API_TOKEN` festlegen, bevor Sie fortfahren!
Navigieren Sie zu `Ihr Repo -> Einstellungen -> CI/CD -> Variablen -> Erweitern -> Variable hinzufügen`.
Der Variablen-Schlüssel sollte `BENCHER_API_TOKEN` und der Variablen-Wert Ihr API-Token sein.
Aktivieren Sie die Kästchen für `Variable schützen` und `Variable maskieren`.
