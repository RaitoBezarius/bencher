Dependendo do seu caso de uso, você pode configurar [Benchmarking Contínuo][continuous benchmarking] no GitLab CI/CD para o seu:

- [Branch de Destino](#target-branch)
- [Merge Requests](#merge-requests)

Certifique-se de ter [criado um token de API][created an api token] e defini-lo como uma variável mascarada nomeada `BENCHER_API_TOKEN` antes de continuar! Navegue para `Seu Repositório -> Configurações -> CI/CD -> Variáveis -> Expandir -> Adicionar variável`. A chave da variável deve ser `BENCHER_API_TOKEN` e o valor da variável deve ser o seu token de API. Marque as caixas de `Proteger variável` e `Mascarar variável`.
