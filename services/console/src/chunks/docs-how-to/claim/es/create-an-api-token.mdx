## Crear un Token de API

Para usar el
<code><a href="/es/docs/explanation/bencher-run/">bencher run</a></code> subcomando de CLI
con la [opción `--token`][token option],
necesitarás crear un token de API.

<h3><a id="add-api-token" href="/console/users/tokens/add?back=aHR0cHM6Ly9iZW5jaGVyLmRldi9kb2NzL2hvdy10by9jbGFpbS8jY3JlYXRlLWFuLWFwaS10b2tlbg==">Haz clic para añadir un Token de API</a></h3>

### Pasos:

1. Navega a la Consola de Bencher.
2. Pasa el cursor sobre tu nombre en la esquina superior derecha.
3. Debería aparecer un menú desplegable. Selecciona `Tokens`.
4. Una vez en la página de Tokens de API, haz clic en el botón `➕ Add`.

[token option]: /es/docs/explanation/bencher-run/#--token-token
