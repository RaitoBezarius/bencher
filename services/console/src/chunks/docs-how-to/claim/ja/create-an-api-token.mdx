## APIトークンを作成する

<code><a href="/ja/docs/explanation/bencher-run/">bencher run</a></code> CLIサブコマンドを
[<code>--token</code>オプション][token option]で使用するには、
APIトークンを作成する必要があります。

<h3><a id="add-api-token" href="/console/users/tokens/add?back=aHR0cHM6Ly9iZW5jaGVyLmRldi9kb2NzL2hvdy10by9jbGFpbS8jY3JlYXRlLWFuLWFwaS10b2tlbg==">APIトークンを追加するにはここをクリック</a></h3>

### 手順:

1. Bencherコンソールに移動します。
2. 右上の自分の名前にマウスを合わせます。
3. ドロップダウンメニューが表示されます。`Tokens`を選択します。
4. APIトークンのページで、`➕ Add`ボタンをクリックします。

[token option]: /ja/docs/explanation/bencher-run/#--token-token
