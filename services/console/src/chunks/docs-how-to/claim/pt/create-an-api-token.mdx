## Criar um Token de API

Para usar o
<code><a href="/pt/docs/explanation/bencher-run/">bencher run</a></code> subcomando CLI
com a [opção `--token`][token option],
você precisará criar um token de API.

<h3><a id="add-api-token" href="/console/users/tokens/add?back=aHR0cHM6Ly9iZW5jaGVyLmRldi9kb2NzL2hvdy10by9jbGFpbS8jY3JlYXRlLWFuLWFwaS10b2tlbg==">Clique para adicionar um Token de API</a></h3>

### Etapas:

1. Navegue até o Console Bencher.
2. Passe o mouse sobre seu nome no canto superior direito.
3. Um menu suspenso deve aparecer. Selecione `Tokens`.
4. Na página de Tokens de API, clique no botão `➕ Add`.

[token option]: /pt/docs/explanation/bencher-run/#--token-token
