## Create an API Token

In order to use the
<code><a href="/docs/explanation/bencher-run/">bencher run</a></code> CLI subcommand
with the [the `--token` option][token option],
you will need to create an API token.

<h3><a id="add-api-token" href="/console/users/tokens/add?back=aHR0cHM6Ly9iZW5jaGVyLmRldi9kb2NzL2hvdy10by9jbGFpbS8jY3JlYXRlLWFuLWFwaS10b2tlbg==">Click to add an API Token</a></h3>

### Steps:

1. Navigate to the Bencher Console.
2. Hover over your name in top right corner.
3. A dropdown menu should appear. Select `Tokens`.
4. Once on the API Tokens page, click the `âž• Add` button.
