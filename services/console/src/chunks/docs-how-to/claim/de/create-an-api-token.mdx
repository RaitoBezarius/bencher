## API-Token erstellen

Um den
<code><a href="/de/docs/explanation/bencher-run/">bencher run</a></code>-CLI-Unterbefehl
mit der [der `--token` Option][token option]
zu verwenden, müssen Sie einen API-Token erstellen.

<h3><a id="add-api-token" href="/console/users/tokens/add?back=aHR0cHM6Ly9iZW5jaGVyLmRldi9kb2NzL2hvdy10by9jbGFpbS8jY3JlYXRlLWFuLWFwaS10b2tlbg==">Klicken Sie hier, um einen API-Token hinzuzufügen</a></h3>

### Schritte:

1. Navigieren Sie zur Bencher-Konsole.
2. Bewegen Sie den Mauszeiger über Ihren Namen in der oberen rechten Ecke.
3. Ein Dropdown-Menü sollte erscheinen. Wählen Sie `Tokens`.
4. Sobald Sie auf der Seite API-Tokens sind, klicken Sie auf die Schaltfläche `➕ Add`.

[token option]: /de/docs/explanation/bencher-run/#--token-token
