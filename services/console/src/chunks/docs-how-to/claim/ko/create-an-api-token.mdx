## API 토큰 생성하기

<code><a href="/ko/docs/explanation/bencher-run/">bencher run</a></code> CLI 하위 명령어를 [the `--token` 옵션][token option]과 함께 사용하려면, API 토큰을 생성해야 합니다.

<h3><a id="add-api-token" href="/console/users/tokens/add?back=aHR0cHM6Ly9iZW5jaGVyLmRldi9kb2NzL2hvdy10by9jbGFpbS8jY3JlYXRlLWFuLWFwaS10b2tlbg==">API 토큰 추가를 위해 클릭하세요</a></h3>

### 단계:

1. Bencher 콘솔로 이동하세요.
2. 오른쪽 상단에 있는 자신의 이름을 마우스로 가리키세요.
3. 드롭다운 메뉴가 나타납니다. `토큰`을 선택하세요.
4. API 토큰 페이지에 도착하면, `➕ Add` 버튼을 클릭하세요.

[token option]: /ko/docs/explanation/bencher-run/#--token-token
