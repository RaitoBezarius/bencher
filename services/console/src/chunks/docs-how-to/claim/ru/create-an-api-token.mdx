## Создание API токена

Для использования
<code><a href="/ru/docs/explanation/bencher-run/">bencher run</a></code> подкоманды CLI
с [опцией `--token`][token option],
вам необходимо создать API токен.

<h3><a id="add-api-token" href="/console/users/tokens/add?back=aHR0cHM6Ly9iZW5jaGVyLmRldi9ydS9kb2NzL2hvdy10by9jbGFpbS8jY3JlYXRlLWFuLWFwaS10b2tlbg==">Кликните, чтобы добавить API токен</a></h3>

### Шаги:

1. Перейдите в Bencher Console.
2. Наведите курсор на ваше имя в правом верхнем углу.
3. Должно появиться выпадающее меню. Выберите `Tokens`.
4. На странице API Tokens нажмите кнопку `➕ Add`.

[token option]: /ru/docs/explanation/bencher-run/#--token-token
