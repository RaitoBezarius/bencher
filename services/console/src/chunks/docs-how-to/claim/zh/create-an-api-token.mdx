## 创建 API 令牌

为了使用 <code><a href="/zh/docs/explanation/bencher-run/">bencher run</a></code> CLI 子命令与 [`--token` 选项][token option]，您需要创建一个 API 令牌。

<h3><a id="add-api-token" href="/console/users/tokens/add?back=aHR0cHM6Ly9iZW5jaGVyLmRldi9kb2NzL2hvdy10by9jbGFpbS8jY3JlYXRlLWFuLWFwaS10b2tlbg==">点击添加 API 令牌</a></h3>

### 步骤：

1. 前往 Bencher 控制台。
2. 将鼠标悬停在右上角的您的名字上。
3. 应出现一个下拉菜单。选择 `Tokens`。
4. 在 API 令牌页面，点击 `➕ Add` 按钮。

[token option]: /zh/docs/explanation/bencher-run/#--token-token
