## Créer un jeton API

Afin d'utiliser la
<code><a href="/fr/docs/explanation/bencher-run/">bencher run</a></code> sous-commande CLI
avec l'[option `--token`][token option],
vous devrez créer un jeton API.

<h3><a id="add-api-token" href="/console/users/tokens/add?back=aHR0cHM6Ly9iZW5jaGVyLmRldi9mci9kb2NzL2hvdy10by9jbGFpbS8jY3JlYXRlLWFuLWFwaS10b2tlbg==">Cliquez pour ajouter un jeton API</a></h3>

### Étapes :

1. Accédez à la console Bencher.
2. Survolez votre nom dans le coin supérieur droit.
3. Un menu déroulant devrait apparaître. Sélectionnez `Tokens`.
4. Une fois sur la page des jetons API, cliquez sur le bouton `➕ Add`.

[token option]: /fr/docs/explanation/bencher-run/#--token-token
