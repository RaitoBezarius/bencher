### Schritt-für-Schritt-Anleitung

Um dieser Schritt-für-Schritt-Anleitung zu folgen, müssen Sie [installieren Sie `git`](https://git-scm.com/downloads), [installieren Sie `cmake`](https://cmake.org/download/#latest), und [installieren Sie die GNU Compiler Collection (GCC) `g++`](https://gcc.gnu.org/install/).

> 🐰 Der Quellcode für diesen Beitrag ist [auf GitHub verfügbar](https://github.com/bencherdev/bencher/tree/main/examples/cpp/google_benchmark).

Erstellen Sie eine C++-Datei mit dem Namen `game.cpp`,
und fügen Sie den Inhalt der obigen FizzBuzz-Implementierung ein.

Verwenden Sie `g++`, um eine ausführbare Datei mit dem Namen `game` zu erstellen, und führen Sie sie dann aus.
