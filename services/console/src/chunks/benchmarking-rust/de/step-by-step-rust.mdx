### Schritt für Schritt folgen

Um dieser ausführlichen Anleitung zu folgen, müssen Sie zuerst [Rust installieren](https://rustup.rs/).

> 🐰 Der Quellcode für diesen Beitrag ist [auf GitHub verfügbar](https://github.com/bencherdev/bencher/tree/main/examples/rust/bench)

Nachdem Sie Rust installiert haben, können Sie ein Terminalfenster öffnen und eingeben: `cargo init game`

Navigieren Sie dann in das neu erstellte Verzeichnis `game`.

Sie sollten ein Verzeichnis namens `src` mit einer Datei namens `main.rs` sehen:

<HelloWorld />

Ersetzen Sie dessen Inhalt mit der oben genannten FizzBuzz-Implementierung. Führen Sie dann `cargo run` aus.
