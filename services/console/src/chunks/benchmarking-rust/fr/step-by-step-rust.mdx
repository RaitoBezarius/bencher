### Suivre étape par étape

Pour suivre ce tutoriel étape par étape, vous devrez [installer Rust](https://rustup.rs/).

> 🐰 Le code source de ce post est [disponible sur GitHub](https://github.com/bencherdev/bencher/tree/main/examples/rust/bench)

Avec Rust installé, vous pouvez alors ouvrir une fenêtre de terminal et entrer : `cargo init game`

Ensuite, naviguez dans le nouveau répertoire `game` créé.
