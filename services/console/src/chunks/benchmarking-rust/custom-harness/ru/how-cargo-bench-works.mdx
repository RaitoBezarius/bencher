## Как работает `cargo bench`

Прежде чем создать собственный тестовый хост,
нам нужно понять, как работают бенчмарки в языке Rust.
Для большинства разработчиков на Rust это означает выполнение команды [команды `cargo bench`][cargo bench].
Команда `cargo bench` компилирует и выполняет ваши бенчмарки.
По умолчанию, `cargo bench` попытается использовать встроенный (но нестабильный) хост libtest bench.
libtest bench затем пройдет через ваш код и выполнит все функции, помеченные атрибутом `#[bench]`.
Чтобы использовать собственный хост бенчмарков, нам нужно указать `cargo bench`, чтобы не использовать libtest bench.

[cargo bench]: https://doc.rust-lang.org/cargo/commands/cargo-bench.html
