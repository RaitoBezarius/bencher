## 编写基准测试代码

在编写性能测试之前，我们需要一些库代码来进行基准测试。对于我们的例子，我们将玩FizzBuzzFibonacci游戏。

FizzBuzzFibonacci的规则如下：

> 编写一个程序，打印从 `1` 到 `100`（包括）的整数：
> - 对于三的倍数，打印 `Fizz`
> - 对于五的倍数，打印 `Buzz`
> - 对于既是三的倍数又是五的倍数，打印 `FizzBuzz`
> - 对于属于斐波那契数列的数字，只打印 `Fibonacci`
> - 对于其他所有数字，打印该数字
