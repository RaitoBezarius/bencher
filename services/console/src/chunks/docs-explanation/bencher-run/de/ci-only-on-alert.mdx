### `--ci-only-on-alert`

<br />

Optional: Beginnen Sie mit dem Veröffentlichen von Ergebnissen in CI, nur wenn [ein Alert generiert wird](/de/docs/explanation/thresholds/#alerts). Wenn ein Alert generiert wird, werden alle nachfolgenden Ergebnisse ebenfalls veröffentlicht, selbst wenn sie keine Alerts enthalten. Erforderlich: `--github-actions`
