### `--hash <HASH>`

<br />

선택사항: 40자리 SHA-1 커밋 해시입니다.
두 보고서가 동일한 branch와 hash를 가지면 동일한 커밋에서 비롯된 것으로 간주됩니다.
따라서 동일한 branch 버전 번호를 가집니다.

제공되지 않은 경우, Bencher CLI는 현재 git 해시를 찾으려고 시도합니다.
그것은 현재 작업 디렉토리에서 git 리포지토리를 찾는 것으로 시작합니다.
성공하지 못하면, 상위 디렉토리로 이동하여 루트 디렉토리까지 모두 다시 시도합니다.
git 리포지토리가 발견되면, 현재 브랜치의 HEAD git 해시가 사용됩니다.
