### `--github-actions <GITHUB_TOKEN>`

<br />

선택 사항: GitHub API 인증 토큰 설정 (예: `--github-actions ${{ secrets.GITHUB_TOKEN }}`).
이 옵션이 설정되고 `bencher run`이 pull request의 일부로 GitHub Actions에서 사용되면,
결과가 pull request에 댓글로 추가됩니다.
그렇지 않으면 결과는 GitHub Check 댓글로 커밋에 추가됩니다.
이 작업을 수행하는 가장 편리한 방법은 [GitHub Actions `GITHUB_TOKEN` 환경 변수](https://docs.github.com/en/actions/security-guides/automatic-token-authentication)입니다.

> 🐰 GitHub Action 내에서 Docker 컨테이너를 실행 중인 경우, 다음 환경 변수를 전달하고 `GITHUB_EVENT_PATH`로 지정된 경로를 마운트해야 합니다:
> - `GITHUB_ACTIONS`
> - `GITHUB_EVENT_NAME`
> - `GITHUB_EVENT_PATH`
> - `GITHUB_SHA`
