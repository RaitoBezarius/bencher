### `--token <TOKEN>`

<br />

선택 사항: `--token` 옵션 또는 `BENCHER_API_TOKEN` 환경 변수를 유효한 API 토큰으로 설정할 수 있습니다.
둘 다 지정된 경우, `--token` 옵션이 `BENCHER_API_TOKEN` 환경 변수보다 우선합니다.
둘 다 지정되지 않은 경우 프로젝트는 `unclaimed` 이어야 합니다.
즉, 이미 프로젝트를 `claimed`했다면 유효한 API 토큰을 제공해야 합니다.
[API 토큰을 생성하려면 여기를 클릭하세요](/console/users/tokens/add?back=L2tvL2RvY3MvZXhwbGFuYXRpb24vYmVuY2hlci1ydW4vIy0tdG9rZW4tdG9rZW4=).
