## `--threshold-measure <MEASURE>`

<br />

Verwenden Sie den angegebenen [Measure][measure]-Namen, Slug oder UUID für einen Threshold.
Wenn der angegebene Wert ein Name oder Slug ist und die Measure noch nicht existiert, wird sie für Sie erstellt.
Wenn der angegebene Wert jedoch eine UUID ist, muss die Measure bereits existieren.

Beispielsweise könnten Sie, um einen Schwellenwert für die Latenz-Messung zu verwenden,
`--threshold-measure latency` schreiben.

[measure]: /de/docs/explanation/benchmarking/#measure
