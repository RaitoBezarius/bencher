## `--threshold-test <TEST>`

<br />

Verwenden Sie den angegebenen Test, um Leistungsregressionen zu erkennen.

Es stehen mehrere verschiedene Tests zur Verfügung:

<ul>
  <li>[Prozentsatz (`percentage`)](#percentage)</li>
  <li>[z-score (`z_score`)](#z-score)</li>
  <li>[t-test (`t_test`)](#t-test)</li>
  <li>[Log-Normal (`log_normal`)](#log-normal)</li>
  <li>[Interquartilsabstand (`iqr`)](#interquartile-range)</li>
  <li>[Delta Interquartilsabstand (`delta_iqr`)](#delta-interquartile-range)</li>
  <li>[Statisch (`static`)](#static)</li>
</ul>

Zum Beispiel, um ein Schwellenwert mit einem Student's t-test zu verwenden,
könnten Sie `--threshold-test t_test` schreiben.
