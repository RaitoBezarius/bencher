## `--threshold-lower-boundary <BOUNDARY>`

<br />

Geben Sie die Untergrenze an.
Die Einschränkungen für die Untergrenze hängen von dem verwendeten [Test][tests] ab.
Entweder die Untergrenze, die Obergrenze oder beide müssen angegeben werden.

Für Details siehe die Dokumentation für den spezifischen Test, den Sie verwenden:

<ul>
  <li>[Statische Untergrenze](#static-lower-boundary)</li>
  <li>[Prozentuale Untergrenze](#percentage-lower-boundary)</li>
  <li>[z-Wert Untergrenze](#z-score-lower-boundary)</li>
  <li>[t-Test Untergrenze](#t-test-lower-boundary)</li>
  <li>[Log-Normal-Untergrenze](#log-normal-lower-boundary)</li>
  <li>[Interquartilsbereich Untergrenze](#interquartile-range-lower-boundary)</li>
  <li>[Delta-Interquartilsbereich Untergrenze](#delta-interquartile-range-lower-boundary)</li>
</ul>

[tests]: #--threshold-test-test
