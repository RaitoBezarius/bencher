## `--threshold-lower-boundary <BOUNDARY>`

<br />

Укажите нижнюю границу.
Ограничения на нижнюю границу зависят от используемого [теста][tests].
Либо нижняя граница, верхняя граница, либо обе должны быть указаны.

Для деталей смотрите документацию по конкретному тесту, который вы используете:

<ul>
  <li>[Процентная нижняя граница](#percentage-lower-boundary)</li>
  <li>[Нижняя граница по z-оценке](#z-score-lower-boundary)</li>
  <li>[Нижняя граница по t-тесту](#t-test-lower-boundary)</li>
  <li>[Нижняя граница логнормального распределения](#log-normal-lower-boundary)</li>
  <li>[Нижняя граница межквартильного размаха](#interquartile-range-lower-boundary)</li>
  <li>[Нижняя граница дельта межквартильного размаха](#delta-interquartile-range-lower-boundary)</li>
  <li>[Статическая нижняя граница](#static-lower-boundary)</li>
</ul>
