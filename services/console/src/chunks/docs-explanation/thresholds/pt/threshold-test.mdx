## `--threshold-test <TEST>`

<br />

Use o Teste especificado para detectar regressões de desempenho.

Existem vários Testes diferentes disponíveis:

<ul>
  <li>[Porcentagem (`percentage`)](#percentage)</li>
  <li>[z-score (`z_score`)](#z-score)</li>
  <li>[t-test (`t_test`)](#t-test)</li>
  <li>[Log Normal (`log_normal`)](#log-normal)</li>
  <li>[Intervalo Interquartil (`iqr`)](#interquartile-range)</li>
  <li>[Delta Intervalo Interquartil (`delta_iqr`)](#delta-interquartile-range)</li>
  <li>[Estático (`static`)](#static)</li>
</ul>

Por exemplo, para usar um Limite com um teste t de Student,
você poderia escrever `--threshold-test t_test`.
