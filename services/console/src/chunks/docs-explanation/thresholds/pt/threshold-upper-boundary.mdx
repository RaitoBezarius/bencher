## `--threshold-upper-boundary <BOUNDARY>`

<br />

Especifique o Limite Superior.
As restrições no Limite Superior dependem do [Teste][tests] utilizado.
Ou o Limite Inferior, o Limite Superior, ou ambos devem ser especificados.

Para detalhes, veja a documentação para o teste específico que você está utilizando:

<ul>
  <li>[Percentage Upper Boundary](#percentage-upper-boundary)</li>
  <li>[z-score Upper Boundary](#z-score-upper-boundary)</li>
  <li>[t-test Upper Boundary](#t-test-upper-boundary)</li>
  <li>[Log Normal Upper Boundary](#log-normal-upper-boundary)</li>
  <li>[Interquartile Range Upper Boundary](#interquartile-range-upper-boundary)</li>
  <li>[Delta Interquartile Range Upper Boundary](#delta-interquartile-range-upper-boundary)</li>
  <li>[Static Upper Boundary](#static-upper-boundary)</li>
</ul>
