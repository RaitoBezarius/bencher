## `--threshold-upper-boundary <BOUNDARY>`

<br />

上限境界を指定します。上限境界の制約は使用する[テスト][tests]によって異なります。下限境界、上限境界、またはその両方を指定する必要があります。

詳細については、ご使用のテストの特定のドキュメントを参照してください：

<ul>
  <li>[静的上限境界](#static-upper-boundary)</li>
  <li>[パーセンテージ上限境界](#percentage-upper-boundary)</li>
  <li>[zスコア上限境界](#z-score-upper-boundary)</li>
  <li>[t検定上限境界](#t-test-upper-boundary)</li>
  <li>[対数正規上限境界](#log-normal-upper-boundary)</li>
  <li>[四分位範囲上限境界](#interquartile-range-upper-boundary)</li>
  <li>[デルタ四分位範囲上限境界](#delta-interquartile-range-upper-boundary)</li>
</ul>

[tests]: #--threshold-test-test
