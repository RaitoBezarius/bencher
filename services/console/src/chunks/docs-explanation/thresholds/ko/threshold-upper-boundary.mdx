## `--threshold-upper-boundary <BOUNDARY>`

<br />

상한선을 지정합니다. 상한선의 제약 조건은 사용되는 [테스트][tests]에 따라 달라집니다. 하한선, 상한선, 또는 둘 다를 지정해야 합니다.

자세한 내용은 사용 중인 특정 테스트의 문서를 참조하십시오:

<ul>
  <li>[퍼센티지 상한선](#percentage-upper-boundary)</li>
  <li>[z-점수 상한선](#z-score-upper-boundary)</li>
  <li>[t-테스트 상한선](#t-test-upper-boundary)</li>
  <li>[로그 정규 상한선](#log-normal-upper-boundary)</li>
  <li>[사분위 범위 상한선](#interquartile-range-upper-boundary)</li>
  <li>[델타 사분위 범위 상한선](#delta-interquartile-range-upper-boundary)</li>
  <li>[Static 상한선](#static-upper-boundary)</li>
</ul>
