## Despliegue

La manera más fácil de desplegar Bencher Self-Hosted es utilizando un tríptico de subcomandos `bencher` CLI:

- [`bencher up`][bencher up]: Levanta los contenedores de Bencher Self-Hosted.
  Esto es similar a ejecutar [`docker compose up`][docker compose up].
- [`bencher logs`][bencher logs]: Muestra la salida de los contenedores de Bencher Self-Hosted,
  si `bencher up` se ejecutó con [la bandera `--detach`][detach flag].
  Esto es similar a ejecutar [`docker compose logs`][docker compose logs].
- [`bencher down`][bencher down]: Detiene y elimina los contenedores de Bencher Self-Hosted,
  si `bencher up` se ejecutó con [la bandera `--detach`][detach flag].
  Esto es similar a ejecutar [`docker compose down`][docker compose down].

[docker compose up]: https://docs.docker.com/reference/cli/docker/compose/up/
[docker compose logs]: https://docs.docker.com/reference/cli/docker/compose/logs/
[docker compose down]: https://docs.docker.com/reference/cli/docker/compose/down/
