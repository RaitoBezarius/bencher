## URL de l'hôte

Lorsque vous utilisez Bencher en auto-hébergement,
vous devrez définir [l'option `--host`][host option]
à l'URL de votre serveur API Bencher
pour la plupart des sous-commandes CLI `bencher`.

Par exemple, si vous utilisiez la sous-commande CLI <code><a href="/fr/docs/explanation/bencher-run/">bencher run</a></code>
