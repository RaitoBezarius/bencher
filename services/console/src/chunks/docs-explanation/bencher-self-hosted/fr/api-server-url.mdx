## URL du serveur API

Lors de l'utilisation de Bencher Self-Hosted,
vous devrez configurer [l'option `--host`][host option]
à l'URL de votre serveur API Bencher
pour la plupart des sous-commandes du CLI `bencher`.

Par exemple, si vous utilisez la sous-commande CLI <code><a href="/fr/docs/explanation/bencher-run/">bencher run</a></code>
avec un serveur API Bencher à `https://api.bencher.example.com`

Si vous exécutez le serveur Bencher Console sur un hôte différent de celui du serveur API Bencher,
vous devrez également configurer [la variable d'environnement `BENCHER_API_URL`][bencher api url environment variable]
sur le serveur Bencher Console pour l'URL distante de votre serveur API Bencher.

Par exemple, si vous utilisez la sous-commande CLI <code><a href="/fr/docs/reference/bencher-compose/#bencher-up">bencher up</a></code>
pour démarrer le serveur Bencher Console avec un serveur API Bencher s'exécutant à `https://api.bencher.example.com`
