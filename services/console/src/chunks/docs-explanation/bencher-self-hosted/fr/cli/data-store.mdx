### `--data-store <DATA_STORE>`

Enregistrez la sauvegarde de la base de données dans ce stockage de données.
Cette opération s'exécute en second.

Stockages de données pris en charge:
- `aws_s3`: Sauvegarde sur AWS S3

Utilisez [la section `database.data_store`](/fr/docs/reference/server-config/#database) de la configuration du serveur API pour configurer le stockage de données pour votre serveur API Bencher auto-hébergé.
