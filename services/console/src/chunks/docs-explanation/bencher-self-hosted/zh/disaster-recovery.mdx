## 灾难恢复

为了防止备份之间的数据丢失，Bencher 提供了内置的灾难恢复解决方案。此功能需要有效的 [Bencher Plus][pricing] 许可证。

支持的复制方案：

- `file`：复制到本地文件路径
- `sftp`：通过 SFTP 复制
- `s3`：复制到任何兼容 S3 的对象存储
- `abs`：复制到 Azure Blob 存储
- `gcs`：复制到谷歌云存储

使用 API 服务器配置中的[ `plus.disaster_recovery` 部分][plus disaster recovery] 为您的 Bencher 自托管 API 服务器设置灾难恢复。
