## 部署

部署 Bencher 自托管的最简单方法是使用一系列 `bencher` CLI 子命令：

- [`bencher up`][bencher up]：启动 Bencher 自托管容器。
  这类似于运行 [`docker compose up`][docker compose up]。
- [`bencher logs`][bencher logs]：查看 Bencher 自托管容器的输出，
  如果 `bencher up` 是以 [`--detach` 标志][detach flag]运行。
  这类似于运行 [`docker compose logs`][docker compose logs]。
- [`bencher down`][bencher down]：停止并移除 Bencher 自托管容器，
  如果 `bencher up` 是以 [`--detach` 标志][detach flag]运行。
  这类似于运行 [`docker compose down`][docker compose down]。

[docker compose up]: https://docs.docker.com/reference/cli/docker/compose/up/
[docker compose logs]: https://docs.docker.com/reference/cli/docker/compose/logs/
[docker compose down]: https://docs.docker.com/reference/cli/docker/compose/down/
