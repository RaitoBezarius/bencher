### `--data-store <DATA_STORE>`

Сохраните резервную копию базы данных в этом хранилище данных.
Эта операция выполняется второй.

Поддерживаемые хранилища данных:
- `aws_s3`: Резервное копирование в AWS S3

Используйте [раздел `database.data_store`][database data store] конфигурации серверного API,
чтобы настроить хранилище данных для вашего самоустановочного API-сервера Bencher.

[database data store]: /ru/docs/reference/server-config/#database
