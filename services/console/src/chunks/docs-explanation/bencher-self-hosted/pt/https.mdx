## HTTPS

Para configurar o HTTPS, você pode usar o Bencher Self-Hosted
atrás de um proxy com HTTPS habilitado
ou utilizar um nome de domínio com um certificado SSL válido.

- [Documentação de HTTPS do Bencher Console][console https]
- [Documentação de `server.tls` do Bencher API][server]
