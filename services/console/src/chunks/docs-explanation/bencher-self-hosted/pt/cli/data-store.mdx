### `--data-store <DATA_STORE>`

Salve o backup do banco de dados neste data store.
Esta operação é executada em segundo lugar.

Data stores suportados:
- `aws_s3`: Backup para AWS S3

Use a [seção `database.data_store`][database data store] da configuração do servidor API
para configurar o data store para o seu servidor API auto-hospedado do Bencher.

[database data store]: /pt/docs/reference/server-config/#database
