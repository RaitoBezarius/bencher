## 배포

Bencher Self-Hosted를 배포하는 가장 쉬운 방법은 `bencher` CLI 하위 명령어의 삼중체를 사용하는 것입니다:

- [`bencher up`][bencher up]: Bencher Self-Hosted 컨테이너를 시작합니다.
  이는 [`docker compose up`][docker compose up]을 실행하는 것과 유사합니다.
- [`bencher logs`][bencher logs]: Bencher Self-Hosted 컨테이너의 출력을 봅니다,
  만약 `bencher up`이 [the `--detach` flag][detach flag]와 함께 실행되었다면.
  이는 [`docker compose logs`][docker compose logs]를 실행하는 것과 유사합니다.
- [`bencher down`][bencher down]: Bencher Self-Hosted 컨테이너를 중지하고 제거합니다,
  만약 `bencher up`이 [the `--detach` flag][detach flag]와 함께 실행되었다면.
  이는 [`docker compose down`][docker compose down]을 실행하는 것과 유사합니다.

[docker compose up]: https://docs.docker.com/reference/cli/docker/compose/up/
[docker compose logs]: https://docs.docker.com/reference/cli/docker/compose/logs/
[docker compose down]: https://docs.docker.com/reference/cli/docker/compose/down/
