## APIサーバーURL

Bencherセルフホスティングを使用する場合、
ほとんどの`bencher` CLIサブコマンドで
Bencher APIサーバーのURLに[`--host`オプション][host option]を設定する必要があります。

例えば、`https://api.bencher.example.com`でBencher APIサーバーを使って
<code><a href="/ja/docs/explanation/bencher-run/">bencher run</a></code> CLIサブコマンドを使用する場合、
