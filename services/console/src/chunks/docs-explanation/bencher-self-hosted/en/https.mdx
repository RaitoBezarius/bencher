## HTTPS

In order to setup HTTPS,
you can use Bencher Self-Hosted behind a proxy with HTTPS enabled
or use a domain name with a valid SSL certificate.

- [Bencher Console HTTPS documentation][console https]
- [Bencher API `server.tls` documentation][server]
