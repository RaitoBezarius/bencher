### `--data-store <DATA_STORE>`

Speichern Sie das Datenbank-Backup in diesem Datenspeicher.
Dieser Vorgang wird als zweites ausgeführt.

Unterstützte Datenspeicher:
- `aws_s3`: Backup zu AWS S3

Verwenden Sie den Abschnitt [„database.data_store“][database data store] der API-Server-Konfiguration,
um den Datenspeicher für Ihren Bencher Self-Hosted API-Server einzurichten.

[database data store]: /de/docs/reference/server-config/#database
