#### Cabeçalho
O Cabeçalho de um Branch é a instância mais recente do Branch.
Ele referencia o Ponto de Início mais recente, se houver um.
Sempre que um Branch recebe um novo Ponto de Início, ele ganha um novo Cabeçalho.
Veja [seleção de branch](/pt/docs/explanation/branch-selection/) para uma visão geral completa.
