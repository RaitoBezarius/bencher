### Branch

Um Branch é a referência do `git` usada ao executar um Relatório (ou seja, nome do branch ou tag). Ao usar o subcomando CLI <code><a href="/pt/docs/explanation/bencher-run/">bencher run</a></code>, se um Branch não for especificado, o nome do branch atual do `git` é usado se estiver disponível. Caso contrário, `main` é usado como o Branch padrão. Consulte [seleção de branch](/pt/docs/explanation/branch-selection/) para uma visão completa.
