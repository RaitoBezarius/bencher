### Medida
Uma Medida é a unidade de medição para uma Métrica.
Por padrão, todos os Projetos começam com uma Medida de `Latência` e `Vazão`
com unidades de `nanosegundos (ns)` e `operações/segundo (ops/s)`, respectivamente.
A Medida é determinada pelo [adaptador do quadro de referência de benchmark](/pt/docs/explanation/adapters/).
