## Relatório
Um Relatório é uma coleção de Benchmarks e suas Métricas para um determinado Branch e Testbed.
Relatórios são mais frequentemente gerados usando o subcomando CLI <code><a href="/pt/docs/explanation/bencher-run/">bencher run</a></code>.
Veja [como rastrear testes de regressão de desempenho](/pt/docs/how-to/track-benchmarks/) para uma visão geral completa.
