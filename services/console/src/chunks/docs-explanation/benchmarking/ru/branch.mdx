### Ветка
Ветка — это `git` ссылка, используемая при запуске Отчета (то есть название ветки или тег). При использовании подкоманды CLI <code><a href="/ru/docs/explanation/bencher-run/">bencher run</a></code>, если Ветка не указана, то используется текущее название `git` ветки, если оно доступно. В противном случае, по умолчанию используется `main`. См. [выбор ветки](/ru/docs/explanation/branch-selection/) для полного обзора.
