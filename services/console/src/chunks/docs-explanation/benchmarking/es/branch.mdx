### Rama
Una Rama es la referencia de `git` utilizada al ejecutar un Informe (es decir, el nombre de la rama o etiqueta). Cuando se utiliza el subcomando CLI <code><a href="/es/docs/explanation/bencher-run/">bencher run</a></code>, si no se especifica una Rama, se usa el nombre de la rama `git` actual si está disponible. De lo contrario, se utiliza `main` como la Rama predeterminada. Consulta [selección de rama](/es/docs/explanation/branch-selection/) para una descripción completa.
