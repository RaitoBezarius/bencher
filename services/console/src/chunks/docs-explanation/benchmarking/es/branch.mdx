### Rama
Una Rama es la referencia de `git` utilizada al ejecutar un Reporte (es decir, el nombre de la rama o la etiqueta).
Por defecto, todos los Proyectos comienzan con una Rama `main`.
Al usar el subcomando de CLI <code><a href="/es/docs/explanation/bencher-run/">bencher run</a></code>,
`main` es la Rama predeterminada si no se proporciona una.
Consulta [selección de ramas](/es/docs/explanation/branch-selection/) para obtener una visión completa.
