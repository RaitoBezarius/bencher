### Branche
Une Branche est la référence `git` utilisée lors de l'exécution d'un Rapport (c'est-à-dire le nom de la branche ou le tag). Par défaut, tous les Projets commencent avec une Branche `main`. Lorsque vous utilisez la sous-commande CLI <code><a href="/fr/docs/explanation/bencher-run/">bencher run</a></code>, `main` est la Branche par défaut si aucune n'est fournie. Voir [sélection de branche](/fr/docs/explanation/branch-selection/) pour un aperçu complet.
