Adapter konvertieren die Ausgabe des Benchmark-Harness in das [Bencher Metric Format (BMF) JSON][bmf].
Die Adapter werden auf dem API-Server ausgeführt, wenn ein neuer Bericht eingeht.
Weitere Details finden Sie in [der Übersicht über das Benchmarking][benchmarking overview].
Ein Adapter kann für [den `bencher run` CLI-Unterbefehl][bencher run]
mit der Option `--adapter` angegeben werden.
Wenn kein Adapter angegeben wird, wird standardmäßig [der `magic` Adapter][magic adapter] verwendet.
