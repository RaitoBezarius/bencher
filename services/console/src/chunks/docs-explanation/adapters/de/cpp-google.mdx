## ➕ C++ Google

Der C++ Google Adapter (`cpp_google`) erwartet Ausgaben von [Google Benchmark](https://github.com/google/benchmark) im [JSON-Format (d.h. `--benchmark_format=json`)](https://github.com/google/benchmark/blob/main/docs/user_guide.md#output-formats).
Die `latency` Messung (d.h. `Nanosekunden (ns)`) wird erfasst.
Nur der Durchschnittswert (d.h. `value`) ist verfügbar. Es gibt keinen `lower_value` und keinen `upper_value`.
