## 🐍 Python ASV

Der Python ASV-Adapter (`python_asv`) erwartet die Ausgabe des [airspeed velocity](https://github.com/airspeed-velocity/asv) CLI-Befehls [asv run](https://asv.readthedocs.io/en/stable/commands.html#asv-run).
Die `latency`-Messung (d.h. `Nanosekunden (ns)`) wird erfasst.
Die `lower_value` und `upper_value` sind das untere und obere Interquartil, jeweils unterhalb und oberhalb des Medians (d.h. `value`).
