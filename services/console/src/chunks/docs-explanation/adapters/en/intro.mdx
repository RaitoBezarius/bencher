Adapters convert benchmark harness output into [Bencher Metric Format (BMF) JSON][bmf].
The adapters run on the API server when a new report is received.
See [the benchmarking overview][benchmarking overview] for more details.
An adapter can be specified for [the `bencher run` CLI subcommand][bencher run]
with the `--adapter` option.
If no adapter is specified, [the `magic` adapter][magic adapter] is used by default.
