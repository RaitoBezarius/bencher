## ➕ C++ Google

El Adaptador de Google C++ (`cpp_google`) espera salida de [Google Benchmark](https://github.com/google/benchmark) en [formato JSON (es decir, `--benchmark_format=json`)](https://github.com/google/benchmark/blob/main/docs/user_guide.md#output-formats).
Se recopila la medida de `latencia` (es decir, `nanosegundos (ns)`).
Solo está disponible el promedio (es decir, `value`). No hay `lower_value` ni `upper_value`.
