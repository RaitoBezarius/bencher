Los adaptadores convierten la salida del arnés de referencia en el formato JSON de Métricas de Bencher ([Bencher Metric Format (BMF) JSON][bmf]).
Los adaptadores se ejecutan en el servidor API cuando se recibe un nuevo informe.
Consulta [el resumen de benchmarking][benchmarking overview] para más detalles.
Un adaptador puede ser especificado para [el subcomando CLI `bencher run`][bencher run]
con la opción `--adapter`.
Si no se especifica un adaptador, [el adaptador `magic`][magic adapter] se utiliza por defecto.
