## 🕸 JavaScript ベンチマーク

JavaScript ベンチマークアダプター（`js_benchmark`）は [Benchmark.js](https://github.com/bestiejs/benchmark.js) の出力を期待しています。
`throughput` 測定 （つまり `operations / second (ops/sec)`）が収集されます。
`lower_value` と `upper_value` はそれぞれ中央値（つまり `value`）より下と上の相対的な誤差の幅です。
