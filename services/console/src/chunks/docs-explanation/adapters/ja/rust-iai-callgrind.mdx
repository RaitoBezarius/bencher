## 🦀 Rust Iai-Callgrind

RustのIaiアダプタ(`rust_iai_callgrind`)は、[Iai-Callgrind](https://github.com/iai-callgrind/iai-callgrind)の出力を期待します。
`instructions`、`l1-hits`、`l2-hits`、`ram-hits`、`total-read-write`、そして`estimated-cycles`の測定値が収集されます。
これらの測定値（つまり`value`）のみが利用可能です。`lower_value`や`upper_value`の測定値はありません。
このアダプタのための測定値は、すべてのプロジェクトに対してデフォルトで作成されるわけではありません。
しかし、このアダプタを使用する場合、これらの測定値はあなたのプロジェクトのために自動的に作成されます。
