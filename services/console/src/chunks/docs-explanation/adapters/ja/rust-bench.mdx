## 🦀 Rust Bench

Rust Bench アダプタ (`rust_bench`) は [libtest bench](https://doc.rust-lang.org/rustc/tests/index.html#benchmarks) の出力を想定しています。
`latency` 測定値（つまり `nanoseconds (ns)`）が収集されます。
`lower_value` と `upper_value` はそれぞれ中央値（つまり `value`）より下と上の偏差です。
