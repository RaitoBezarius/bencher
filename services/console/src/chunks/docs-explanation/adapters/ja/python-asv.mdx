## 🐍 Python ASV

Python ASV アダプター (`python_asv`) は [airspeed velocity](https://github.com/airspeed-velocity/asv) CLI [asv run](https://asv.readthedocs.io/en/stable/commands.html#asv-run) の出力を想定しています。
`latency` 測定値（つまり `ナノ秒 (ns)`）が収集されます。
`lower_value` と `upper_value` は、それぞれ中央値（つまり `value`）の下と上の四分位範囲です。
