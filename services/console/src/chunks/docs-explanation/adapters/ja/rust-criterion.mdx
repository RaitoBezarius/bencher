## 🦀 Rust Criterion

Rust Criterionアダプター (`rust_criterion`) は[Criterion](https://github.com/bheisler/criterion.rs)の出力を期待しています。
`latency` 測定（つまり `ナノ秒 (ns)`）が収集されます。
`lower_value` と `upper_value` は、利用可能な場合は傾き、そうでない場合は平均（つまり `value`）のそれぞれ下限と上限です。
