## ➕ C++ 구글

C++ 구글 어댑터(`cpp_google`)는 [Google Benchmark](https://github.com/google/benchmark) 출력을 [JSON 형식(`--benchmark_format=json`을 통해)](https://github.com/google/benchmark/blob/main/docs/user_guide.md#output-formats)으로 기대합니다.
`지연 시간` 측정(즉, `나노초(ns)`)이 수집됩니다.
평균(즉, `값`)만 사용 가능합니다. `lower_value`와 `upper_value`는 없습니다.
