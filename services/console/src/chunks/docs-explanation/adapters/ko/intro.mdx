어댑터는 벤치마크 하네스 출력을 [Bencher Metric Format (BMF) JSON][bmf]으로 변환합니다.
새 보고서가 수신될 때 API 서버에서 어댑터가 실행됩니다.
자세한 내용은 [벤치마킹 개요][benchmarking overview]를 참조하세요.
어댑터는 `--adapter` 옵션으로 [the `bencher run` CLI 서브커맨드][bencher run]에 대해 지정할 수 있습니다.
어댑터가 지정되지 않은 경우 기본적으로 [the `magic` 어댑터][magic adapter]가 사용됩니다.
