## 🪄 Magie <small>(par défaut)</small>

L'adaptateur Magic (`magic`) est un superset de tous les autres adaptateurs.
Pour cette raison, il est l'adaptateur par défaut pour `bencher run`.
Cependant, l'`adaptateur magic` doit être utilisé uniquement pour l'exploration.
