## ♦️ Benchmark Ruby

L'adaptateur Benchmark Ruby (`ruby_benchmark`) attend la sortie du [module Benchmark](https://github.com/ruby/benchmark) pour les méthodes `#bm`, `#bmbm` et `#benchmark`.
Un libellé est requis pour chaque benchmark.
La mesure de `latence` (c'est-à-dire `nanosecondes (ns)`) est recueillie.
Seule la valeur rapportée (c'est-à-dire `valeur`) est disponible. Il n'y a ni `valeur_inférieure` ni `valeur_supérieure`.
