## ➕ C++ Google

L'adaptateur C++ Google (`cpp_google`) nécessite un résultat [Google Benchmark](https://github.com/google/benchmark) au format [JSON (c'est-à-dire `--benchmark_format=json`)](https://github.com/google/benchmark/blob/main/docs/user_guide.md#output-formats).
La mesure de `latence` (c'est-à-dire `nanosecondes (ns)`) est recueillie.
Seule la moyenne (c'est-à-dire `valeur`) est disponible. Il n'y a ni `valeur_inférieure` ni `valeur_supérieure`.
