## 🦀 Rust Criterion

Rust Criterion 适配器（`rust_criterion`）期待 [Criterion](https://github.com/bheisler/criterion.rs) 的输出。
会收集 `latency` 测量（即 `纳秒 (ns)`）。
`lower_value` 和 `upper_value` 分别是斜率（如果可用）或平均值（如果不可用）的下限和上限（即 `值`）。
