适配器将基准测试工具的输出转换为 [Bencher Metric Format (BMF) JSON][bmf]。当API服务器收到一份新报告时，适配器会运行。更多细节请参见[基准测试概览][benchmarking overview]。可以为[`bencher run` CLI 子命令][bencher run]通过`--adapter`选项指定一个适配器。如果没有指定适配器，那么默认使用[ `magic` 适配器][magic adapter]。
