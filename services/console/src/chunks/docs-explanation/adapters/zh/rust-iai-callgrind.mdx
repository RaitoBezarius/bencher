## 🦀 Rust Iai-Callgrind

Rust Iai 适配器（`rust_iai_callgrind`）期望 [Iai-Callgrind](https://github.com/iai-callgrind/iai-callgrind) 的输出。
它收集 `instructions`、`l1-hits`、`l2-hits`、`ram-hits`、`total-read-write` 和 `estimated-cycles` 这几个度量值。
只有这些度量值（即 `value`）是可用的。没有 `lower_value` 和 `upper_value` 这样的度量值。
对于这个适配器，这些度量值不会为所有项目默认创建。
然而，当你使用这个适配器时，这些度量值会自动为你的项目创建。
