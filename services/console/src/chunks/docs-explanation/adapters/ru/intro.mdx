Адаптеры преобразуют выходные данные инструментальной среды тестирования производительности в JSON формате [Bencher Metric Format (BMF)][bmf].
Адаптеры работают на сервере API, когда получен новый отчет.
Смотрите [обзор тестирования производительности][benchmarking overview] для получения более подробной информации.
Адаптер можно указать для подкоманды CLI [the `bencher run`][bencher run] с опцией `--adapter`.
Если адаптер не указан, по умолчанию используется [адаптер `magic`][magic adapter].
