## 🐍 Python ASV

O Adaptador Python ASV (`python_asv`) espera a saída do CLI [airspeed velocity](https://github.com/airspeed-velocity/asv) [asv run](https://asv.readthedocs.io/en/stable/commands.html#asv-run).
A Medida de `latência` (ou seja, `nanossegundos (ns)`) é coletada.
Os valores `lower_value` e `upper_value` são o intervalo interquartílico abaixo e acima da mediana (ou seja, `value`), respectivamente.
