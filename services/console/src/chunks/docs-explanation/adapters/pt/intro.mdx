Os adaptadores convertem a saída do harness de benchmark para o formato JSON do [Bencher Metric Format (BMF)][bmf].
Os adaptadores são executados no servidor da API quando um novo relatório é recebido.
Veja [a visão geral de benchmarking][benchmarking overview] para mais detalhes.
Um adaptador pode ser especificado para [o subcomando CLI `bencher run`][bencher run]
com a opção `--adapter`.
Se nenhum adaptador for especificado, [o adaptador `magic`][magic adapter] é usado por padrão.
