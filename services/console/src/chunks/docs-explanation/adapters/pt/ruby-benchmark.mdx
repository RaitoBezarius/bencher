## ♦️ Benchmark do Ruby

O Adaptador de Benchmark do Ruby (`ruby_benchmark`) espera a saída do [módulo Benchmark](https://github.com/ruby/benchmark) para os métodos `#bm`, `#bmbm`, e `#benchmark`.
Uma etiqueta é necessária para cada benchmark.
A Medida de `latência` (ou seja, `nanosegundos (ns)`) é coletada.
Apenas o valor reportado (ou seja, `valor`) está disponível. Não existem `valor_inferior` e `valor_superior`.
