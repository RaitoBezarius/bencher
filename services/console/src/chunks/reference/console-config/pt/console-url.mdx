## URL do Console

Por padrão, [o servidor da API Bencher][server config] espera que o servidor do Console Bencher esteja rodando em `http://localhost:3000`.
Se o seu servidor do Console Bencher estiver rodando em outro lugar, então você precisa configurar
[a opção `console.url`][console url].
