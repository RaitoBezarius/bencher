## 콘솔 URL

기본적으로 [Bencher API 서버][server config]는 Bencher Console 서버가 `http://localhost:3000`에서 실행 중이라고 기대합니다.
만약 Bencher Console 서버가 다른 곳에서 실행 중이라면, [`console.url` 옵션을 설정][console url]해야 합니다.

[server config]: /ko/docs/reference/server-config/
[console url]: /ko/docs/reference/server-config/#console
