The Bencher Console UI no requiere ninguna configuración. Sin embargo, algunos valores predeterminados se pueden personalizar para satisfacer sus necesidades utilizando las siguientes variables de entorno:

- `BENCHER_API_URL`: La URL para [el servidor API de Bencher][server config]. Por defecto, está configurado en `http://localhost:61016`.
- `INTERNAL_API_URL`: Cuando se utiliza una red basada en Docker, como con `docker compose`, esto es necesario para la comunicación Docker-a-Docker al usar un `BENCHER_API_URL` local. No lo configure cuando utilice un `BENCHER_API_URL` remoto, como `https://bencher.example.com`. Por defecto, esto no está configurado.
- `GITHUB_CLIENT_ID`: La ID del Cliente de GitHub para una [Aplicación GitHub Autoalojada][github app] para habilitar "Iniciar sesión con GitHub". Esto requiere un [plan Bencher Plus Enterprise][bencher plus enterprise]. Por defecto, esto no está configurado.
