## URL de la Consola

Por defecto, [el servidor API de Bencher][server config] espera que el servidor de la Consola Bencher esté ejecutándose en `http://localhost:3000`. Si tu servidor de la Consola Bencher se está ejecutando en otro lugar, entonces necesitas configurar
[la opción `console.url`][console url].
