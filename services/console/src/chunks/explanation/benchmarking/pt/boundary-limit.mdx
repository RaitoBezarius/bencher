#### Limite de Fronteira
Um Limite de Fronteira é o valor calculado a partir de um Limite Inferior ou Superior.
Ele é usado para comparar com uma nova Métrica.
Um Alerta é gerado quando uma nova Métrica está abaixo de um Limite de Fronteira Inferior ou acima de um Limite de Fronteira Superior.
Veja [limiares](/pt/docs/explanation/thresholds/) para uma visão completa.
