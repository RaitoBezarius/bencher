### Branch
Um Branch é a referência `git` usada ao executar um Relatório (ou seja, nome do branch ou tag). Por padrão, todos os Projetos começam com um Branch `main`. Ao usar o subcomando da CLI <code><a href="/pt/docs/explanation/bencher-run/">bencher run</a></code>, `main` é o Branch padrão se nenhum for fornecido. Consulte [seleção de branch](/pt/docs/explanation/branch-selection/) para uma visão geral completa.
