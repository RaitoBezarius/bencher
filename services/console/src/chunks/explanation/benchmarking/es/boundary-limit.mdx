#### Límite de Frontera
Un Límite de Frontera es el valor calculado a partir de una Frontera Inferior o una Frontera Superior.
Se utiliza para comparar con una nueva Métrica.
Se genera una Alerta cuando una nueva Métrica está por debajo de un Límite de Frontera Inferior o por encima de un Límite de Frontera Superior.
Consulta [umbrales](/es/docs/explanation/thresholds/) para una descripción completa.
