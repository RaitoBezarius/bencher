#### Punto de Inicio
Una Rama puede tener un [Punto de Inicio][start point]. Un Punto de Inicio es otra Rama en una versión específica (y `git` hash, si está disponible). Las Métricas Históricas y, opcionalmente, los Umbrales se copian desde el Punto de Inicio. Consulta [selección de ramas](/es/docs/explanation/branch-selection/) para obtener una visión completa.
