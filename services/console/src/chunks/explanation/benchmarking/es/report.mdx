## Reporte
Un Reporte es una colección de Benchmarks y sus Métricas para una Rama y Banco de Pruebas particulares.
Los Reportes se generan con mayor frecuencia utilizando el subcomando de la CLI <code><a href="/es/docs/explanation/bencher-run/">bencher run</a></code>.
Consulta [cómo seguir las pruebas de regresión de rendimiento](/es/docs/how-to/track-benchmarks/) para una visión completa.
