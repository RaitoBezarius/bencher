### Ветка
Ветка — это ссылка `git`, используемая при запуске Отчета (например, имя ветки или тег).
По умолчанию все проекты начинаются с Ветки `main`.
При использовании подкоманды CLI <code><a href="/ru/docs/explanation/bencher-run/">bencher run</a></code>,
`main` является веткой по умолчанию, если другая не указана.
См. [выбор ветки](/ru/docs/explanation/branch-selection/) для полного обзора.
