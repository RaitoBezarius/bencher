## Отчет
Отчет представляет собой набор Контрольных показателей и их Метрик для определенной Ветки и Испытательного стенда.
Отчеты чаще всего генерируются с использованием подкоманды CLI <code><a href="/ru/docs/explanation/bencher-run/">bencher run</a></code>.
Смотрите [как отслеживать регрессионные тесты производительности](/ru/docs/how-to/track-benchmarks/) для полного обзора.
