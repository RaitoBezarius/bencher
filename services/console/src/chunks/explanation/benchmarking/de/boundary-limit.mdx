#### Grenzenwert
Ein Grenzenwert ist der Wert, der aus einer unteren oder oberen Grenze berechnet wird. Er wird verwendet, um ihn mit einem neuen Metrikwert zu vergleichen. Eine Warnmeldung wird generiert, wenn ein neuer Metrikwert unter einem unteren Grenzenwert oder über einem oberen Grenzenwert liegt. Siehe [Schwellenwerte](/de/docs/explanation/thresholds/) für einen vollständigen Überblick.
