#### Start Point
ブランチは[スタートポイント][start point]を持つことができます。
スタートポイントは特定のバージョン（および利用可能であれば`git`のハッシュ）の別のブランチです。
スタートポイントから履歴メトリクスと、オプションでスレッショルドがコピーされます。
詳細については[ブランチ選択](/ja/docs/explanation/branch-selection/)をご参照ください。

[start point]: https://git-scm.com/docs/git-branch#Documentation/git-branch.txt-ltstart-pointgt
