## `--threshold-measure <MEASURE>`

<br />

Используйте указанное [Measure][measure] имя, слаг или UUID для порога.
Если указанное значение представляет собой имя или слаг и Measure еще не существует, оно будет создано для вас.
Однако, если указанное значение является UUID, то Measure уже должен существовать.

Например, чтобы использовать порог для Measure задержки,
вы могли бы написать `--threshold-measure latency`.

[measure]: /ru/docs/explanation/benchmarking/#measure
