## `--threshold-measure <MEASURE>`

<br />

Utilisez le nom, le slug ou l'UUID du [Mesure][measure] spécifié pour un Seuil.
Si la valeur spécifiée est un nom ou un slug et que le Mesure n'existe pas déjà, il sera créé pour vous.
Cependant, si la valeur spécifiée est un UUID, le Mesure doit déjà exister.

Par exemple, pour utiliser un Seuil pour le Mesure de Latence,
vous pourriez écrire `--threshold-measure latency`.

[measure]: /fr/docs/explanation/benchmarking/#measure
