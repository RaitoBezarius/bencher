## `--threshold-test <TEST>`

<br />

Utilisez le Test spécifié pour détecter les régressions de performance.

Il existe plusieurs Tests différents disponibles :

<ul>
  <li>[Statique (`static`)](#static)</li>
  <li>[Pourcentage (`percentage`)](#percentage)</li>
  <li>[score-z (`z_score`)](#z-score)</li>
  <li>[test-t (`t_test`)](#t-test)</li>
  <li>[Log Normal (`log_normal`)](#log-normal)</li>
  <li>[Écart Interquartile (`iqr`)](#interquartile-range)</li>
  <li>[Écart Interquartile Delta (`delta_iqr`)](#delta-interquartile-range)</li>
</ul>

Par exemple, pour utiliser un Seuil avec un test t de Student,
vous pourriez écrire `--threshold-test t_test`.
