## `--threshold-max-sample-size <SAMPLE_SIZE>`

<br />

Opcionalmente, especifique el número máximo de [Métricas][metric] utilizadas para ejecutar una Prueba.
Si este máximo se excede, se ignorarán las Métricas más antiguas.
El tamaño de muestra especificado debe ser mayor o igual a `2`.
Si también se establece [la opción `--threshold-min-sample-size`][threshold min sample size option],
entonces el tamaño de muestra especificado debe ser mayor o igual a `--threshold-min-sample-size`.
Esta opción no se puede usar con [la Prueba Estática (`static`)][static].

Por ejemplo, para usar un Umbral con un tamaño de muestra máximo de `100`,
podría escribir `--threshold-max-sample-size 100`.
Si hubiera más de `100` Métricas, solo se incluirían las `100` Métricas más recientes.
Por el contrario, si hubiera `100` o menos Métricas, todas las Métricas serían incluidas.
