## `--threshold-measure <MEASURE>`

<br />

Utilice el nombre del [Measure][measure] especificado, slug o UUID para un Umbral.
Si el valor especificado es un nombre o slug y el Measure no existe ya, se creará para usted.
Sin embargo, si el valor especificado es un UUID, entonces el Measure ya debe existir.

Por ejemplo, para usar un Umbral para el Measure de Latencia,
podría escribir `--threshold-measure latency`.

[measure]: /es/docs/explanation/benchmarking/#measure
