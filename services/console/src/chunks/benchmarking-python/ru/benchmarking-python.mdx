## Бенчмаркинг в Python

Два популярных варианта для бенчмаркинга в Python это:
[pytest-benchmark](https://pytest-benchmark.readthedocs.io)
и
[airspeed velocity (asv)](https://github.com/airspeed-velocity/asv)

`pytest-benchmark` — это мощный инструмент для бенчмаркинга,
интегрированный с популярным [фреймворком для тестирования `pytest`](https://docs.pytest.org/en/stable/).
Он позволяет разработчикам измерять и сравнивать производительность их кода, выполняя бенчмарки наряду с модульными тестами.
Пользователи могут легко сравнивать результаты своих бенчмарков локально
и экспортировать свои результаты в различных форматах, таких как JSON.

`airspeed velocity (asv)` — это еще один продвинутый инструмент для бенчмаркинга в экосистеме Python.
Одним из ключевых преимуществ `asv` является его способность генерировать детализированные и интерактивные HTML-отчеты,
что упрощает визуализацию трендов производительности и выявление регрессий.
Кроме того, `asv` поддерживает [относительный непрерывный бенчмаркинг](/ru/docs/how-to/track-benchmarks/#relative-continuous-benchmarking) из коробки.
