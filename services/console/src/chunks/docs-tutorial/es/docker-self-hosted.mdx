## Bencher Autohospedado

Bencher es de código abierto y puede ser autohospedado.
Si estás interesado en usar Bencher Cloud, consulta el [tutorial de Inicio Rápido de Bencher Cloud](/es/docs/tutorial/quick-start/).
Este tutorial te guiará para configurar Bencher Autohospedado con Docker.

> - [Configuración del Servidor de Consola](/es/docs/reference/console-config/)
