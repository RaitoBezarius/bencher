Bencher ist Open Source und selbst hostbar.
Wenn Sie daran interessiert sind, Bencher Cloud zu nutzen, schauen Sie sich das [Bencher Cloud Schnellstart-Tutorial](/de/docs/tutorial/quick-start/) an.
Dieses Tutorial wird Ihnen helfen, Bencher Self-Hosted mit Docker einzurichten.

> 🐰 Sobald Sie sich bei der Verwendung von Bencher Self-Hosted wohl fühlen, sollten Sie die folgenden Ressourcen in Betracht ziehen:
> - [Bencher API Server Konfiguration](/de/docs/reference/server-config/)
> - [Replikation zu AWS S3 mit Litestream](https://litestream.io/guides/s3/)
> - [Bencher Architektur](/de/docs/reference/architecture/)
