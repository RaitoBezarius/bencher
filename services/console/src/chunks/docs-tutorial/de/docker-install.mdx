## Docker installieren

Um die UI- und API-Server in diesem Tutorial auszuführen, müssen Sie `docker` installiert haben.
Überprüfen Sie, ob Sie `docker` installiert haben.

Führen Sie aus: `docker --version`

Sie sollten etwas Ähnliches sehen:

<DockerVersionOutput />

Es ist in Ordnung, wenn Ihre Versionsnummer anders ist. Es ist nur wichtig, dass dieser Befehl funktioniert.
Wenn nicht, folgen Sie den [Anweisungen zur Installation von `docker`](https://docs.docker.com/get-docker/).
