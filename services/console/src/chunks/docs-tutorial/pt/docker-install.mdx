## Instale o Docker

Para executar os servidores de UI e API neste tutorial, você precisará ter o `docker` instalado.
Verifique se você tem o `docker` instalado.

Execute: `docker --version`

Você deve ver algo como:

<DockerVersionOutput />

Tudo bem se o número da sua versão for diferente. O importante é que este comando funcione.
Se não, siga as [instruções para instalar o `docker`](https://docs.docker.com/get-docker/).
