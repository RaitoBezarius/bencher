## Bencher Self-Hosted

Bencher é open source e pode ser auto-hospedado.
Se você está interessado em usar o Bencher Cloud, confira o [tutorial de Início Rápido do Bencher Cloud](/pt/docs/tutorial/quick-start/).
Este tutorial irá configurá-lo para usar o Bencher Self-Hosted com Docker.
