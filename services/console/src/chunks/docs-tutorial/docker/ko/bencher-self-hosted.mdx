## Bencher 독립 호스팅

Bencher는 오픈 소스이며 독립 호스팅이 가능합니다.
Bencher Cloud를 사용하고자 한다면, [Bencher Cloud 빠른 시작 튜토리얼][quick start]을 확인하세요.
이 튜토리얼은 Docker를 사용하여 Bencher 독립 호스팅을 설정하는 방법을 제공합니다.

> 🐰 Bencher 독립 호스팅을 사용하여 익숙해지신 후에는 다음 리소스를 확인해 보세요:
> - [Bencher 독립 호스팅][bencher self hosted]
> - [콘솔 서버 구성][console server config]
> - [API 서버 구성][api server config]
> - [Bencher 아키텍처][bencher architecture]
