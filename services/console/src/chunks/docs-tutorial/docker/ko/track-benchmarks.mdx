## 벤치마크 추적하기

이제 벤치마크 결과를 추적할 준비가 되었습니다!
이를 위해 [`bencher run` CLI 서브커맨드][bencher run]를 사용하여 벤치마크를 실행하고 결과를 수집합니다.
다음 명령을 실행하세요:

<BencherRun isConsole={false} />

설정에 맞춰 벤치마크 명령어를 수정해야 할 수도 있습니다.
아직 벤치마크가 없다면, 벤치마크 명령어로 [`bencher mock` 서브커맨드][bencher mock]를 사용하여 모의 데이터를 생성할 수 있습니다.
모든 것이 예상대로 작동한다면, 출력의 마지막 부분은 다음과 유사할 것입니다:

<BencherRunOutput />

이제 브라우저에서 각 벤치마크에 대한 결과를 확인할 수 있습니다.
`View results`의 링크를 클릭하거나 복사하여 붙여넣으세요.
프로젝트를 [클레임하려면][claim project], `Claim this project` 링크를 브라우저에 클릭하거나 복사하여 붙여넣으세요.
