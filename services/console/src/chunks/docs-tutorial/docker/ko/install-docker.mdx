## Docker 설치

이 튜토리얼에서 콘솔과 API 서버를 실행하려면 `docker`가 설치되어 있어야 합니다.
`docker`가 설치되어 있는지 확인하세요. 다음 명령어를 실행하세요:

<TerminalCommand code="docker --version" />

다음과 같은 결과가 표시됩니다:

<DockerVersionOutput />

버전 번호가 다르더라도 괜찮습니다. 이 명령어가 작동하는 것이 중요합니다.
작동하지 않는다면 [Docker 설치 지침][install docker]을 따르세요.

[install docker]: https://docs.docker.com/get-docker/
