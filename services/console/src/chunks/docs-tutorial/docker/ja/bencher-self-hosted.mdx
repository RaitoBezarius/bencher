## Bencher Self-Hosted

Bencherはオープンソースで、セルフホストが可能です。
Bencher Cloudの使用に興味がある場合は、[Bencher Cloud クイックスタートチュートリアル][quick start]をチェックしてください。
このチュートリアルでは、Dockerを使用してBencher Self-Hostedのセットアップを行います。

> - [コンソールサーバーの設定][console server config]
> - [APIサーバーの設定][api server config]
