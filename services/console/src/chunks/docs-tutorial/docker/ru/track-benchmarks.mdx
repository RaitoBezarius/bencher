## Отслеживание ваших бенчмарков

Теперь вы готовы отслеживать результаты ваших бенчмарков!
Для этого вы будете использовать подкоманду CLI [«bencher run»][bencher run],
чтобы запускать ваши бенчмарки и собирать результаты.
Запустите:

<BencherRun isConsole={false} />

Возможно, вам потребуется изменить команду бенчмарка, чтобы она соответствовала вашей настройке.
Если у вас еще нет бенчмарков, вы можете просто использовать подкоманду [«bencher mock»][bencher mock] в качестве вашей команды бенчмарка, чтобы сгенерировать фиктивные данные.
Если всё работает как ожидается, конец вывода должен выглядеть примерно так:

<BencherRunOutput />

Теперь вы можете просмотреть результаты для каждого из ваших бенчмарков в браузере.
Нажмите или скопируйте и вставьте ссылки из «Просмотреть результаты».
Чтобы [присвоить проект][claim project], нажмите или скопируйте и вставьте ссылку «Присвоить этот проект» в ваш браузер.
