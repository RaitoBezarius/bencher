## Bencher Auto-hébergé

Bencher est open source et auto-hébergeable.
Si vous êtes intéressé par l'utilisation de Bencher Cloud, consultez le [tutoriel de démarrage rapide de Bencher Cloud][quick start].
Ce tutoriel vous expliquera comment configurer Bencher Auto-hébergé avec Docker.

> - [Configuration du serveur Console][console server config]
