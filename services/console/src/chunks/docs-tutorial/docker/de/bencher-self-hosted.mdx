## Bencher Selbstgehostet

Bencher ist Open Source und selbst hostbar.
Wenn Sie Bencher Cloud verwenden möchten, schauen Sie sich das [Bencher Cloud Schnellstart-Tutorial][quick start] an.
Dieses Tutorial hilft Ihnen dabei, Bencher Selbstgehostet mit Docker einzurichten.
