## Bencher Self-Hosted

O Bencher é de código aberto e pode ser hospedado por você mesmo.
Se você estiver interessado em usar o Bencher Cloud, confira o [tutorial de início rápido do Bencher Cloud][quick start].
Este tutorial irá te orientar na configuração do Bencher Self-Hosted com Docker.

> 🐰 Quando se sentir confortável usando o Bencher Self-Hosted, considere verificar os seguintes recursos:
> - [Bencher Self-Hosted][bencher self hosted]
> - [Configuração do Console Server][console server config]
> - [Configuração do API Server][api server config]
> - [Arquitetura do Bencher][bencher architecture]
