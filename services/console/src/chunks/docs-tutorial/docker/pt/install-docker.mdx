## Instalar o Docker

Para executar os servidores Console e API neste tutorial, você precisará ter o `docker` instalado.
Verifique se você tem o `docker` instalado. Execute:

<TerminalCommand code="docker --version" />

Você deve ver algo como:

<DockerVersionOutput />

Não tem problema se o número da sua versão for diferente. O importante é que este comando funcione.
Caso contrário, siga as [instruções para instalar o `docker`][install docker].

[install docker]: https://docs.docker.com/get-docker/
