## Bencher Autohospedado

Bencher es de código abierto y se puede hospedar por uno mismo.
Si estás interesado en usar Bencher Cloud, consulta el [tutorial de inicio rápido de Bencher Cloud][quick start].
Este tutorial te guiará para configurarlo usando Bencher Autohospedado con Docker.

> 🐰 Una vez que te sientas cómodo usando Bencher Autohospedado, considera revisar los siguientes recursos:
> - [Bencher Autohospedado][bencher self hosted]
> - [Configuración del Servidor de Consola][console server config]
> - [Configuración del Servidor API][api server config]
