## Instalar Docker

Para ejecutar los servidores de consola y API en este tutorial, necesitarás tener `docker` instalado. Comprueba si tienes `docker` instalado. Ejecuta:

<TerminalCommand code="docker --version" />

Deberías ver algo como:

<DockerVersionOutput />

Está bien si tu número de versión es diferente. Es importante que este comando funcione. Si no es así, sigue las [instrucciones para instalar `docker`][install docker].

[install docker]: https://docs.docker.com/get-docker/
