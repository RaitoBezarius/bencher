## 安装 Docker

为了在本教程中运行 UI 和 API 服务器，你需要安装 `docker`。
检查你是否已经安装了 `docker`。

运行：`docker --version`

你应该会看到类似这样的信息：

<DockerVersionOutput />

即使你的版本号不同也没关系，关键是这个命令能够正常工作。
如果不能，请按照[安装 `docker` 的说明](https://docs.docker.com/get-docker/)进行操作。
