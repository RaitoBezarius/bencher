> 🐰 如果你遇到错误提示 `running scripts is disabled on this system`：
> - `以管理员身份运行` 打开 PowerShell
> - 运行：`Set-ExecutionPolicy -ExecutionPolicy RemoteSigned`
> - 输入：`Y`
> - 重新运行此脚本
