## 벤치마크 추적하기

이제 벤치마크 결과를 추적할 준비가 되었습니다!
이를 위해 [‘bencher run’ CLI 서브커맨드][bencher run]를 사용하여 벤치마크를 실행하고 결과를 수집합니다. 다음을 실행하세요:

<BencherRun isConsole={true} />

설정에 맞게 벤치마크 명령어를 수정해야 할 수도 있습니다.
아직 벤치마크가 없는 경우, 벤치마크 명령어로 [‘bencher mock’ 서브커맨드][bencher mock]를 사용하여 모의 데이터를 생성할 수 있습니다.
모든 것이 예상대로 작동하면 출력의 끝은 다음과 비슷할 것입니다:

<BencherRunOutput />

이제 브라우저에서 각 벤치마크의 결과를 볼 수 있습니다.
`View results`의 링크를 클릭하거나 복사하여 붙여넣으세요.
[프로젝트를 클레임하려면][claim project], `Claim this project` 링크를 클릭하거나 복사하여 브라우저에 붙여넣으세요.
