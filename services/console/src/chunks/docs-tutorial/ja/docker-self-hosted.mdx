Bencher はオープンソースでセルフホスト可能です。
Bencher Cloud を使用することに興味があるなら、[Bencher Cloud クイックスタートチュートリアル](/ja/docs/tutorial/quick-start/)を確認してください。
このチュートリアルでは、Dockerを使用して Bencher セルフホストの設定を行います。
