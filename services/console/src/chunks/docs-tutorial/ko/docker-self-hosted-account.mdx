## Bencher Self-Hosted 계정 생성

<h3><a href="http://localhost:3000/auth/signup" target="_blank">Bencher Self-Hosted에 가입하기</a></h3>
<br />

다음 주소로 이동하여 로컬 Bencher Self-Hosted 인스턴스에 계정을 생성합니다: http://localhost:3000/auth/signup

계정을 생성한 후, `bencher up`을 실행한 터미널 창으로 돌아가십시오.
다음과 같은 내용을 볼 수 있어야 합니다:

<DockerSignupOutput />

브라우저에서 `Confirm Email` 링크로 이동하거나 `Confirmation Token`을 다음 주소의 `Confirm Token` 필드에 복사하십시오: http://localhost:3000/auth/confirm

그 후에, Bencher Self-Hosted 계정에 로그인 되어 있어야 합니다!
