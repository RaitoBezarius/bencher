## Docker 설치하기

이 튜토리얼에서 UI와 API 서버를 실행하려면 `docker`가 설치되어 있어야 합니다.
`docker`가 설치되어 있는지 확인해 보세요.

다음을 실행: `docker --version`

다음과 같은 것을 볼 수 있어야 합니다:

<DockerVersionOutput />

버전 번호가 다르다 하더라도 괜찮습니다. 중요한 것은 이 명령어가 동작한다는 것입니다.
그렇지 않다면 [`docker` 설치 지침](https://docs.docker.com/get-docker/)을 따르세요.
