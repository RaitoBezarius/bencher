## Bencher 셀프 호스티드

Bencher는 오픈 소스이며 자체 호스팅이 가능합니다. Bencher Cloud 사용에 관심이 있다면 [Bencher Cloud 빠른 시작 튜토리얼](/ko/docs/tutorial/quick-start/)을 확인해 보세요. 이 튜토리얼은 Docker를 사용하여 Bencher 셀프 호스티드를 설정하는 방법을 안내합니다.
