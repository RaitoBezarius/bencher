## Bencher Autohébergé

Bencher est open source et autohébergeable.
Si vous êtes intéressé par l'utilisation de Bencher Cloud, consultez le [tutoriel de démarrage rapide de Bencher Cloud](/fr/docs/tutorial/quick-start/).
Ce tutoriel vous guidera dans la configuration de Bencher Autohébergé avec Docker.

> - [Configuration du serveur Console](/fr/docs/reference/console-config/)
