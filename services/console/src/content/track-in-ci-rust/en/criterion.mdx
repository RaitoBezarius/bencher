## Steps for Bencher Cloud

1. [Create a Bencher Cloud account][bencher sign up].
2. [Create an API token][console users tokens add] and add it to your CI as a secret.
3. Create a workflow for your CI, like [GitHub Actions][github actions] or [GitLab CI/CD][gitlab ci cd].
4. [Install the Bencher CLI][install bencher cli] in your CI workflow.
5. Run your benchmarks with the
   <code><a href="/docs/explanation/bencher-run/">bencher run</a></code>
   subcommand in your CI workflow using
   [the `rust_criterion` adapter][adapter rust_criterion].

## Steps for Bencher Self-Hosted

1. Create a [Bencher Self-Hosted instance][bencher self hosted].
2. Create an account on your Bencher Self-Hosted instance.
3. Create an API token and add it to your CI as a secret.
4. Create a workflow for your CI, like [GitHub Actions][github actions] or [GitLab CI/CD][gitlab ci cd].
5. [Install the Bencher CLI][install bencher cli] in your CI workflow.
   Make sure the CLI version matches the version of your Bencher Self-Hosted instance.
6. Run your benchmarks with the
   <code><a href="/docs/explanation/bencher-run/">bencher run</a></code>
   subcommand in your CI workflow using
   [the `rust_criterion` adapter][adapter rust_criterion]
   and setting [the `--host` option][host option] to your Bencher Self-Hosted instance URL.
